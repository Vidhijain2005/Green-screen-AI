# Green Screen AI

Green Screen AI is a web application that uses AI-powered background removal to automatically convert uploaded images into green-screen-ready assets. The tool is designed to simplify the editing workflow for video editors, content creators, and designers by eliminating the need for manual background removal.

## Features

* Upload JPG, PNG, and WEBP images
* AI-powered background removal
* Automatic green screen replacement
* Original and processed image preview
* High-quality image download
* Clean and responsive user interface
* Fast and efficient image processing

## How It Works

1. Upload an image.
2. The AI detects the main subject and removes the background.
3. The removed background is replaced with a pure green screen.
4. Preview the processed image.
5. Download the final image for use in video editing software.
